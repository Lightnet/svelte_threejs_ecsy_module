# svelte_threejs_ecsy_module

## Information:
 Build for threejs, ecsy, and svelte as well other packages to be module design.

## Packages / Libs:
 * ecsy
 * threejs
 * svelte
 * gun
 * gulp
 * polka

## Setup:
 Note this is work in progress.

```
  npm install
```
 It will install the pacakges.

```
  gulp default
```
 Will build and run web server.
